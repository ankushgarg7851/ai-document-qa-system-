
# AI Document Q&A System

##  Overview
This project allows users to upload a PDF and ask questions. The system extracts text and provides answers based on the document content.

##  Features
- Upload PDF
- Extract text
- Ask questions
- Get answers

##  Tech Stack
- Python
- Streamlit
- PyPDF2
- OpenAI API

##  How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py
